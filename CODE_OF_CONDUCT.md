# Code of Conduct

## Our Commitment

Blockiy is committed to providing a welcoming, respectful, and inclusive environment for everyone participating in this project.

We encourage constructive discussions, collaboration, and knowledge sharing within the Cardano community.

## Expected Behavior

Participants are expected to:

- Be respectful and professional
- Welcome different viewpoints and experiences
- Accept constructive feedback
- Focus on improving the project and helping others

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment or discrimination
- Personal attacks
- Offensive language
- Spam or malicious activity
- Publishing confidential information without permission

## Enforcement

Project maintainers reserve the right to remove comments, discussions, or contributions that violate this Code of Conduct.

If you experience or witness unacceptable behavior, please contact:
