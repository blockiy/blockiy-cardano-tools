# Usage Guide

The Cardano Pool KEY / KES Expiry Checker is available as a live tool on the official Blockiy website.

## How to Use

1. Open the tool:
   https://www.blockiy.com/cardano-pool-key-expiry-checker/

2. Follow the on-page instructions.

3. Use the generated information to review your Cardano stake pool KEY and KES expiry status.

## Notes

This tool is designed for Cardano stake pool operators and educational use.

Always verify critical operational data with your own Cardano node and official Cardano CLI tools.
