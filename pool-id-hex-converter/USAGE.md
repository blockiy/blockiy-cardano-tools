# Usage Guide

The Cardano Pool ID ↔ HEX Converter is available as a live tool on the official Blockiy website.

## How to Use

1. Open the tool:
   https://www.blockiy.com/cardano-pool-id-hex-converter/

2. Enter either a Cardano Pool ID or its hexadecimal representation.

3. The tool automatically converts it to the corresponding format.

## Notes

This tool is intended for Cardano stake pool operators, developers, and community members.

Always verify critical data before using it in production environments.
